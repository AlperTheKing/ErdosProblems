## Statement A: Overload Collapse

For a gamma-minimal connected-B maximum cut, let

```text
R(P) = sum_{v in P} T(v)
```

for a shortest row `P in cyc[f]`, where `L=ell(f)`.

Claim:

```text
R(P) > L*N
```

implies the pentagonal global case:

```text
L = 5,
the positive-K component generated by row co-membership from P is V(G),
and all bad edges in that component have length 5.
```

This is the "overload collapse" part. It should be proved before the
pentagonal stability inequality is used.

## Statement B: Pentagonal Matrix Stability

In the global length-5 case, define

```text
I(P) = sum_{g in M} (1/|cyc[g]|) * sum_{Q in cyc[g]} |P cap Q|.
```

Then

```text
75*(I(P)-N) <= 2*(N^2 - 25*|M|).
```

Equivalently, with `Over(P)=5*(I(P)-N)` and
`Def=N^2-25*|M|`,

```text
15*Over(P) <= 2*Def.
```

## Abstract Five-Layer Version

For a five-layer model

```text
A0 -- A1 -- A2 -- A3 -- A4
```

with bad edges only in `A4 x A0`, a distinguished row
`x0-x1-x2-x3-x4`, and every bad edge having at least one length-4
layered B-path, set

```text
s_P =
  d_M(x0) + d_M(x4)
  + sum_{i=1}^3 sum_{ab in M_4}
      #(layered shortest paths ab through x_i) / Z_ab.
```

The matrix stability target is

```text
s_P <= N + (2/75)*(N^2 - 25*m).
```

The current abstract gates use full cut-domination
`delta_M(S) <= delta_B(S)` for all `S`. They do not yet encode the
gamma-tie condition. If a cut-domination-only counterexample appears, the
next test is whether gamma-tie removes it.

Important caveat: the pure five-layer model is not yet the full actual
pentagonal overload theorem. In the N=10 equality atom, the row interval
layers have sizes `[1,2,2,2,1]`, so only 8 of the 10 vertices lie in the
row interval. The two remaining vertices are still active: they are
endpoints of two off-interval bad edges whose shortest rows pass through
the distinguished row. Thus the actual PMS theorem needs an extended
pentagonal attachment model, not only bad edges in `A4 x A0`.

## Proof Obligations

1. Prove overload collapse:

```text
R(P)>L*N => L=5 and the row lives in a global length-5 K-component.
```

The likely mechanism is that any long or non-global overload row admits a
prefix/terminal-shadow descent, contradicting gamma-minimality.

2. Prove pentagonal matrix stability:

```text
s_P - N <= (2/75)*(N^2 - 25*m).
```

The equality atom suggests a sparse pentagonal bottleneck theorem: every
unit of row capture above the clean C5-blow-up AM-GM bound must consume at
least `3/2` units of bad-edge deficit from `N^2/25`.

The proof target should include the extended attachment signatures above.
The pure five-layer theorem is a useful subcase, but it does not account
for the equality atom by itself.

3. Identify equality cases:

```text
balanced C5 blow-ups,
and the N=10 Petersen-like sparse atom above.
```

The proof should explain why non-uniform blow-ups that violate PMS are not
maximum cuts.


## Weighted Equality-Atom Blow-Up Stress

The equality atom `I?BD@g]Qo` was also tested as a weighted blow-up at the
10 base vertices. Since the blow-up maxcut objective is multilinear in the
class-side variables, it is enough to check the `2^10` quotient side
assignments to decide whether the inherited cut is maximum.

Uniform weights `t=1,2,3,4,5` keep the inherited cut maximum and produce
overloaded rows. The PMS margin is zero only at `t=1`; for larger uniform
blow-ups it is positive.


This suggests a sharper stability phenomenon inside the equality-atom
family: once the inherited cut is maximum, row overload may force a highly
symmetric/uniform weighting. Nonuniform quotient-maximal perturbations seem
to destroy overload rather than create PMS counterexamples.

### Fixed-core endpoint reduction for the seven-cut PMS target

In the seven-cut weighted equality-atom target, the denominator/path constants depend only on the six core weights

```text
(w0,w3,w4,w5,w6,w8).
```

The endpoint weights

```text
(w1,w2,w7,w9)
```

enter only through

```text
s = w1+w2+w7+w9,
m = w1*w9 + w2*w7 + w7*w9,
```

and the internal capture terms

```text
c19*w1*w9 + c27*w2*w7 + c79*w7*w9,
```

where `c19=A19/Z19`, `c27=A27/Z27`, `c79=A79/Z79` are fixed by the core.
Thus, for fixed core, the PMS margin is a four-variable quadratic/rational endpoint minimization under the endpoint constraints

```text
w5 >= w9,
w6 >= w7,
w3+w5 >= w2+w9,
w4+w6 >= w1+w7,
w1*w9+w2*w7+w7*w9 <= A,
<= B,
<= C,
```

where

```text
A=w0*w6+w3*w8+w5*w8,
B=w0*w5+w3*w8+w5*w8,
C=w0*w6+w4*w8+w6*w8.
```

Exact gate `_codex_ocpms_core_endpoint_opt.py` enumerated all core weights in `[1,4]^6` and all feasible integer endpoints for each core:

```text
worst endpoint minimum = 0 at all-ones
no negative endpoint optimum
```

This suggests the proof should split into:

1. Prove the fixed-core endpoint quadratic minimization is nonnegative under the seven endpoint constraints.
2. Prove equality in that endpoint problem forces the all-ones core (or is paid by core slack).

This is a smaller proof target than the raw 10-variable rational implication.

### Core coefficient-deficit positivity

For the seven-cut endpoint reduction, write core variables

```text
a=w0, b=w3, c=w4, d=w5, e=w6, h=w8,
A=a-1, B=b-1, C=c-1, D=d-1, E=e-1, H=h-1.
```

The internal coefficients satisfy the universal caps

```text
c19 <= 7/3,
c27 <= 7/3,
c79 <= 2,
```

with equality at the all-ones core. More usefully, the deficit numerators become coefficient-positive after shifting by 1:

```text
(7/3-c19) numerator =
7*A*D*E + 4*A*D + 4*A*E + A
+ 7*C*D*H + 4*C*D + 4*C*H + C
+ 7*D*E*H + 11*D*E + 11*D*H + 9*D
+ 4*E*H + 5*E + 5*H.
```

```text
(7/3-c27) numerator =
7*A*D*E + 4*A*D + 4*A*E + A
+ 7*B*E*H + 4*B*E + 4*B*H + B
+ 7*D*E*H + 11*D*E + 4*D*H + 5*D
+ 11*E*H + 9*E + 5*H.
```

```text
(2-c79) numerator =
2*A*D*E + A*D + A*E
+ 2*B*C*H + B*C
+ 2*B*E*H + B*E + 3*B*H + B
+ 2*C*D*H + C*D + 3*C*H + C
+ 2*D*E*H + 3*D*E + 3*D*H + 2*D
+ 3*E*H + 2*E + 4*H.
```

All coefficients are nonnegative and constants are zero. Thus any core nonuniformity creates an explicit positive reservoir by lowering at least one dangerous endpoint coefficient from its extremal all-ones value. The crude universal-cap PMS bound is false, but the actual margin is exactly that crude bound plus these positive coefficient-deficit payments multiplied by the corresponding endpoint products.

### 2026-06-30 Scale Guardrail And KKT-Leaf Route

The seven-cut implication is false for arbitrary positive real weights unless
the natural class-size normalization is imposed:

```text
w_i >= 1
```

or, equivalently for the intended graph model, the `w_i` are positive integer
class sizes.  On the all-equal ray `w_i=t`, the cut inequalities hold, but

```text
m = 3t^2,        N = 10t,
I = (32/3)t,
```

so the PMS target becomes

```text
50t <= 50t^2,
```

which fails for every `0<t<1` and is tight at `t=1`.  All real-weight
statements below must therefore be read with `w_i>=1`.

For the fixed-core endpoint proof, rename

```text
a=w0, b=w3, c=w4, p=w5, q=w6, r=w8,
x=w1, y=w2, u=w7, v=w9.
```

Let

```text
C = a+b+c+p+q+r,       s = x+y+u+v,
D27 = ap+br+pr,        D19 = aq+cr+qr,
D0  = aq+br+pr,
D79 = apq+bcr+bqr+cpr+pqr.
```

With

```text
alpha = A27/(q D27),
beta  = A19/(p D19),
gamma = A79/D79,
rho   = 50+75 alpha,
sigma = 50+75 beta,
tau   = 50+75 gamma,
```

the PMS margin is equivalent to minimizing

```text
F = 2(C+s)^2 + 75C - rho*y*u - sigma*x*v - tau*u*v
```

over

```text
x,y,u,v >= 1,
u <= q,       v <= p,
x+u <= c+q,  y+v <= b+p,
m=xv+yu+uv <= D0, D27, D19.
```

The endpoint feasible region is the product of two quadrilaterals,

```text
P_L={(x,u): x>=1, u>=1, u<=q, x+u<=c+q},
P_R={(y,v): y>=1, v>=1, v<=p, y+v<=b+p}.
```

Thus a continuous minimum lies on one of `9 x 9` endpoint face pairs, with
either no active quadratic cap or one active cap among `D0,D27,D19`.  On each
leaf the KKT equations are:

```text
T^T grad(F)=0
```

or, on an active cap branch,

```text
T^T(grad(F)+theta grad(m))=0,   m=Dj.
```

Dropping the sign condition `theta>=0` only enlarges the algebraic candidate
set, so the proof can close leafwise by denominator-clearing and shifted
positivity.  The first hard leaf to test is

```text
x=y=1,       u+v+uv = Dj,
```

with tangent condition

```text
(H-rho-tau*v)(1+u) = (H-sigma-tau*u)(1+v),
H = 4(C+2+u+v).
```

This KKT-leaf route replaces the failed coordinate/subset descent and the
failed low-degree coefficientwise LP route.  The coefficient reservoirs
`Delta27`, `Delta19`, and `Delta79` remain the intended positive shifted
terms that pay for core nonuniformity after substituting the leaf equations.
