## Definitions

For a gamma-minimal connected-`B` maximum cut, let `M` be the bad edges,
`ell(g)` the shortest odd-cycle length of `g`, `cyc[g]` the set of shortest
`B`-geodesic rows closing `g`, and

```text
T(v) = sum_g ell(g) * Pr_{Q in cyc[g]}[v in Q].
```

For a row `P in cyc[f]`, set

```text
L = ell(f),
R(P) = sum_{v in P} T(v).
```

The row is overloaded when `R(P) > L*N`.

## Target Lemma 1: OC Collapse

If `R(P) > L*N`, then:

```text
L = 5,
the positive K-component generated by shortest-cycle co-occurrence from P is V(G),
and every bad edge in that component has length 5.
```

Equivalently, all genuine row overload collapses to a global pentagonal
length-5 core.

## Target Lemma 2: PMS Matrix Stability

In the collapsed length-5 case define

```text
I(P) = sum_{g in M} (1/|cyc[g]|) * sum_{Q in cyc[g]} |V(P) cap V(Q)|.
```

Since every bad edge has length `5`, this is

```text
I(P) = sum_{v in P} mu(v),    mu(v)=T(v)/5.
```

The target inequality is

```text
75*(I(P)-N) <= 2*(N^2 - 25*|M|).
```

Equivalently,

```text
15*Over(P) <= 2*(N^2 - 25*|M|),
Over(P)=5*(I(P)-N).
```

Local exact gate result:

```text
pms_rows=36
pms_fail=0
min_margin=0
```

The zero-margin atom is:

```text
g6 = I?BD@g]Qo
N = 10
P = (7,5,8,6,9)
f = (7,9)
M = ((1,9),(2,7),(7,9))
I(P) = 32/3
N^2 - 25|M| = 25
margin = 0
```

This is the sparse pentagonal/Petersen-like equality atom.

## Proof Hinge

The desired proof can be split cleanly:

1. **Collapse proof.** Show that any row overload admits gamma-minimality
   rotations unless it is global, length-5, and pentagonal.
2. **PMS proof.** Prove the 5-layer stability theorem.  Complete weighted
   `C5` blow-ups satisfy `I(P) <= N`; overload can only come from sparse path
   bottlenecks, and the target says every unit of bottleneck excess costs at
   least `3/2` bad edges below the `N^2/25` extremal count.

The equality atom `I?BD@g]Qo` should be the sharp sparse bottleneck model.
