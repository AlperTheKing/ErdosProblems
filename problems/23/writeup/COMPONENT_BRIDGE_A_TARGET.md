# Component-Local BRIDGE-A Proof Target

Status: proof-facing note for the load-PSC withdrawal bank.

The current exact route reduces the prehalf bank to the following inequality.
For load levels `0=t_0<...<t_r`, write

```text
w_j = t_{j+1}-t_j,
H_j = {v : T(v)>t_j},
sigma_j = delta_B(H_j)-delta_M(H_j),
eta = N^2/25 - |M|.
```

Define

```text
A_j = w_j*(N+eta-t_j-t_{j+1})*|H_j|,
B_j = 25*A_j - N*w_j*sigma_j,
A_pre = sum_{2*t_j<N} A_j,
W_neg = sum_{2*t_j>=N and B_j<0} (-B_j).
```

The validated global BRIDGE-A statement is now

```text
W_neg <= 15*A_pre.
```

Claude exact-verified this on the full standing battery. The exact maximum ratio is

```text
W_neg/A_pre = 8305/559 ~ 14.85689,
```

so `15` is the minimal integer coefficient. The coefficient `12` is false on the standing battery.

## Layer-cake form

Let `tau` be the upper endpoint of the last band with `2*t_j<N`, and let

```text
u_v = min(T(v), tau).
```

Claude exact-verified the identity

```text
A_pre = sum_v u_v*(N+eta-u_v).
```

Thus the global statement is equivalently a high-withdrawal bound by a single
truncated parabolic volume.

## K-component localization

Let K-components be generated by all shortest B-geodesic cycle paths. For a
component `C`, define

```text
A_pre(C) = sum_{2*t_j<N} w_j*(N+eta-t_j-t_{j+1})*|H_j cap C|.
```

For each high negative band, split the withdrawal proportionally by component
volume in that superlevel:

```text
D_C += (-B_j) * |H_j cap C| / |H_j|.
```

The exact full-battery result confirmed by Claude is

```text
D_C <= 15*A_pre(C)
```

for every component. The exact maximum component ratio is again

```text
D_C/A_pre(C) = 8305/559,
```

so the component-local target has the same minimal integer coefficient as the global target.

## Failed simplifications

The following stronger reductions are false and should not be used:

```text
W_neg <= c*N*Xi(min(T,tau))       for c <= 5,
```

because some witnesses have `Xi=0` but `W_neg>0`.

Pointwise load-value allocation is also false:

```text
R(T_v) <= 20*u_v*(N+eta-u_v).
```

The first witnessed failure has source `2000` and demand `4000`.

Same-high-superlevel localization is false: a high negative band may require
source from outside its own `H_j` and outside its high suffix. Therefore the
component-local proportional split is currently the smallest surviving local
form.

## Analytic target

For each K-component `C`, prove a corridor/bank allocation sending the
proportional high withdrawals `D_C` into the prehalf parabolic bank
`A_pre(C)`. The proof must use K-component geometry; scalar pressure at the
half cutoff and pointwise vertex reserves are insufficient.

## Component-share dominance

A further exact gate supports the following structural explanation. For every
high negative band `j` and K-component `C`,

```text
|H_j cap C| / |H_j| <= A_pre(C) / A_pre.
```

Equivalently,

```text
|H_j cap C| * A_pre <= |H_j| * A_pre(C).
```

A local full-battery scan (`_bridgea_share.py`) found `252741` exact rows,
`0` violations, minimum margin `0`. If this lemma is proved, then the
component-local coefficient follows formally from the global coefficient:

```text
D_C = sum_j (-B_j)*|H_j cap C|/|H_j|
    <= (A_pre(C)/A_pre) * W_neg.
```

Thus global `W_neg <= c*A_pre` implies component-local
`D_C <= c*A_pre(C)` for the same coefficient `c`.


## Stronger monotone-share candidate

A stronger local scan found that the component-share dominance appears to hold
bandwise before averaging. For every high negative band `j`, every prehalf band
`k` with positive bank coefficient, and every K-component `C`,

```text
|H_j cap C| / |H_j| <= |H_k cap C| / |H_k|.
```

Equivalently,

```text
|H_j cap C| * |H_k| <= |H_k cap C| * |H_j|.
```

Since `A_pre(C)/A_pre` is a weighted average of the prehalf shares
`|H_k cap C|/|H_k|`, this monotone-share lemma implies component-share dominance
immediately. Local exact scan `_bridgea_share_monotone_probe.py` found
`947960` comparisons through census `N<=11` plus the structured battery used in
that script, with `0` violations.

Proof interpretation: high-load negative bands are not allowed to be more
K-component-concentrated than any prehalf superlevel. This is a sharper and more
combinatorial statement than the weighted component bank inequality.

## Active-component containment candidate

A still sharper scan suggests that the monotone-share lemma is explained by a
support containment fact.

For every high negative band `j`, the superlevel `H_j` is contained in a single
K-component `C_j`. Moreover, every prehalf band `k` with positive bank
coefficient also satisfies

```text
H_k subseteq C_j.
```

Thus for such `(j,k,C)`, the monotone-share inequality is trivial:

```text
|H_j cap C_j|/|H_j| = 1 = |H_k cap C_j|/|H_k|,
```

and all other components have share `0` on the high band.

Claude exact-verified `_active_comp.py` on the full standing battery:

```text
high negative bands = 143931
prehalf comparisons = 553704
violations = 0
```

This includes Myc(Grotzsch) N=23, glued graphs, and k-lane cases.
Proof target: show that a high negative BRIDGE-A band can occur only when all
positive prehalf load is carried by the same K-component. This is likely a
max-cut/connected-B consequence of `B_j<0`, rather than a numerical accident.

## Positive-support K-connectivity candidate

The active-component containment appears to reduce further. If any high negative
BRIDGE-A band exists, then the entire positive-load support

```text
{v : T(v)>0}
```

is contained in one K-component. Since every superlevel `H_j` is a subset of
`{T>0}`, this immediately implies active-component containment and hence the
monotone-share lemma.

Local exact scan `_bridgea_positive_component_probe.py` found

```text
high-negative configurations = 78225
violations = 0
```

on the structured/family battery plus census `N<=11`.

This is now the cleanest support target: prove that a high negative BRIDGE-A
band cannot coexist with two positive K-components.
