# ALLX-PSC50 Variational Strengthening

This note records a stronger form of the Perron square-coarea certificate.
It is **not proved** and the current checks are only scouts, but it removes
the algebraic Perron-vector dependency from the statement.

## Statement

Fix a connected-`B` maximum cut with bad-edge set `M`.  Let `P[v,f]=p_f(v)`.
For any nonzero vector `x_f >= 0` on `M`, define

```text
phi(v) = sum_f x_f p_f(v),
L      = sum_v phi(v)^2,
h(v)  = N phi(v)^2 / L.
```

Let

```text
Xi_x = sum_{uv in B} |h(u)-h(v)| - sum_{uv in M} |h(u)-h(v)|.
```

The proposed strengthening is

```text
(ALLX-PSC50)   L + |M| + Xi_x/50 <= N + N^2/25.
```

Equivalently, after multiplying by `L>0`,

```text
L^2 - (N + N^2/25 - |M|) L
  + (N/50) * (sum_B |phi_u^2-phi_v^2| - sum_M |phi_u^2-phi_v^2|)
<= 0.
```

This is homogeneous in `x`.

## Relation To PSC-50

If `x` is a nonnegative Perron vector of `O=P^T P`, normalized by `|x|_2=1`,
then `L = x^T O x = rho(O)` on the selected top component.  Therefore
ALLX-PSC50 implies PSC-50:

```text
rho(O) + |M| + Xi(h)/50 <= N + N^2/25.
```

Since max-cut cut-domination gives `Xi(h)>=0` by coarea, PSC-50 implies
`rho(O)+|M| <= N+N^2/25`, i.e. the SBC certificate.

## Why This Form May Be Easier

The Perron-only PSC-50 has algebraic data: `lambda`, `x`, and `h` require
verified eigenvalue/eigenvector handling.  ALLX-PSC50 is a finite-dimensional
inequality over the nonnegative cone generated by the geodesic columns of `P`.
For rational `x`, all quantities are rational once the absolute-value signs
are fixed.

So a proof can target a homogeneous variational inequality:

```text
phi in cone(P), phi != 0
  => ||phi||_2^2 + |M| + (N/(50||phi||_2^2)) * Xi(phi^2)
     <= N + N^2/25.
```

The cost is that this is strictly stronger than PSC-50 and still implies the
full theorem.

## Balanced C5[t] Closed Form

For the balanced `C5[t]` cut with the bad block between parts `4` and `0`,
write `x` as a nonnegative `t x t` matrix on bad edges.  With `||x||_F=1`,
let row sums be `r_i`, column sums `c_j`, and total `s`.

Then

```text
L = sum_i r_i^2 + sum_j c_j^2 + 3s^2/t.
```

The normalized `h` is:

```text
h(part 4 vertex i) = 5t r_i^2 / L,
h(part 0 vertex j) = 5t c_j^2 / L,
h(middle parts)    = 5s^2 / (tL).
```

The ALLX-PSC50 inequality reduces to

```text
L + Xi_x/50 <= 5t.
```

Uniform `x_ij=1/t` gives equality: `L=5t` and `Xi_x=0`.
Random stress did not find a nonuniform violation; this suggests a Cauchy
stability inequality at the extremal quotient.

## Caveat On The CUT-PSC50 Split

The auxiliary inequality

```text
|M| + Xi(h)/50 <= N^2/25
```

is useful language for the cut-pressure budget, but it is not a cheap
sublemma: since max-cut cut-domination gives `Xi(h)>=0`, this inequality
already implies `|M|<=N^2/25`. Thus CUT-PSC50 is itself theorem-strength.

The advantage of ALLX-PSC50 is different: it packages the cut-pressure term
and the spectral term into one homogeneous inequality over `phi=Px`, avoiding
separate algebraic Perron-vector handling.

## Correction: Normalization Is Required

ALLX-PSC50 must be stated with either `||x||_2=1` or with the Rayleigh
quotient

```text
lambda_x = ||P x||_2^2 / ||x||_2^2.
```

The scale-free form is:

```text
for every nonzero x>=0,
  lambda_x + |M| + Xi_x/50 <= N + N^2/25,
```

where

```text
phi = P x,
L = ||phi||_2^2,
h(v) = N phi(v)^2 / L,
Xi_x = TV_B(h)-TV_M(h).
```

Equivalently, after choosing the normalization `||x||_2=1`, this is the
statement above with `L=lambda_x`. The earlier multiplied form was written as
if arbitrary scaling were allowed; that was incorrect. With arbitrary scaling,
`L` can be made arbitrarily large while `h` is unchanged.

## Order-Cell Quadratic Form

Fix a vector `phi=Px` and assume the signs of all differences
`phi(u)^2-phi(v)^2` on graph edges are fixed. Then

```text
D(phi^2) := sum_B |phi_u^2-phi_v^2| - sum_M |phi_u^2-phi_v^2|
```

is a diagonal quadratic form in `phi`:

```text
D(phi^2) = sum_v c_v phi(v)^2 = phi^T C phi,
```

where each edge contributes `+sgn(phi_u^2-phi_v^2)` to one endpoint and the
opposite sign to the other endpoint, with the sign reversed for bad edges.
Thus on one order cell,

```text
D(phi^2) = x^T P^T C P x.
```

With the normalization `||x||_2=1`, ALLX-PSC50 is equivalent on that cell to

```text
L + (N/(50L)) R <= A,

L = x^T O x,
R = x^T P^T C P x,
A = N + N^2/25 - |M|.
```

After multiplying by `L>0`:

```text
L^2 - A L + (N/50)R <= 0.
```

So a sufficient cell certificate is

```text
x^T P^T C P x <= (50/N) (x^T O x)(A - x^T O x)
```

for every `x>=0` in the cell.  The max-cut CD condition gives only
`D(phi^2)>=0` by coarea; the missing upper bound must use the bad-count deficit
and the triangle-free geodesic structure.  This is the local matrix form of the
PSC proof problem.

## Perron-Is-Worst Is False

A random scan found a small counterexample to the auxiliary principle that
`F(x)=lambda_x+Xi_x/50` is maximized at Perron.

Graph/cut:

```text
graph6: GCQb`o
N=8, cut side 01110000
M={(0,5),(4,7)}, both ell=5
O = [[9/2,2],[2,9/2]]
```

Perron direction `(1,1)` gives

```text
lambda=13/2,
Xi=20/13,
F=13/2 + (20/13)/50 = 6.5307692307.
```

A local one-dimensional optimization over `x=(cos theta, sin theta)` gives

```text
x ~= (0.69292973, 0.72100513),
F ~= 6.532357345 > F(Perron).
```

The normalized ALLX-PSC50 margin remains positive:

```text
N+N^2/25-|M|-F ~= 2.027642655.
```

Thus ALLX-PSC50 is genuinely stronger than PSC-50; it cannot be reduced to
PSC by a Perron-is-worst variational principle.

## Coefficient 25 Scout

The original PSC-50 used the correction `Xi/50`.  A local scout suggests the
natural coefficient may be `25`:

```text
PSC-25:  lambda + |M| + Xi/25 <= N + N^2/25,
CUT-25:  |M| + Xi/25 <= N^2/25.
```

Floating evidence on census `N<=10` plus two-lane and dense k-lane hard cases:

```text
PSC-25 violations: 0
CUT-25 violations: 0
```

The visible worst case is the same `N=8` graph with graph6 string `G?chr96Fchr96w`, cut index 8, with

```text
N=8, |M|=2, lambda=7, Xi=96/7.
```

For CUT this gives

```text
(N^2/25-|M|)/Xi = (14/25)/(96/7) = 49/1200,
```

so the required coefficient is `1200/49 ~= 24.4898`, just below `25`.
This points to `25` as the natural constant coming from `ell(f)^2>=25`.
Claude has been asked to exact-test PSC-25 and CUT-25.
